## Issue 1 — `detect_beat_class_pattern()` Slices into Non-Overlapping Pairs/Triplets; Pattern at Offset ≠ 0 Is Missed

**Severity: REQUIRED**

**Location:** Section 3, `detect_beat_class_pattern()`:

```python
# Bigeminy: alternating normal and PVC
if len(classes) >= 6:
    pairs = [(classes[i], classes[i+1]) for i in range(0, len(classes)-1, 2)]
    if all(a == 0 and b == 1 for a, b in pairs):
        return "BIGEMINY_PVC"
```

**Two flaws:**

**Flaw A — Wrong slice:** `range(0, len(classes)-1, 2)` generates indices `[0, 2, 4, 6, ...]`. For `classes = [0, 1, 0, 1, 0, 1]` (true bigeminy starting at index 0), `pairs = [(0,1), (0,1), (0,1)]` → correctly returns `BIGEMINY_PVC`. But for `classes = [0, 0, 1, 0, 1, 0, 1]` (first beat normal, then bigeminy starts at beat 2), `pairs = [(0,0), (1,0), (1,0)]` → first pair `(0,0)` fails the check → returns `None`. The bigeminy is missed.

Real ECG recordings rarely start exactly on a bigeminal cycle boundary. A patient with bigeminy who happened to have a normal beat at the start of the 10-second window would be reported as `OCCASIONAL_ECTOPY`.

**Flaw B — Stride-2 slicing on non-paired data:** For `classes = [0, 1, 0, 1, 0, 1, 0]` (7 beats), `range(0, 6, 2)` gives `[0, 2, 4]` → pairs `[(0,1), (0,1), (0,1)]`. The last beat (index 6) is not checked. This produces a false positive for incomplete bigeminy.

**Required fix:** Use a sliding window scan (not stride-2 slicing) to detect the pattern at any offset. Require the pattern to hold for ≥ 4 consecutive pairs:

```python
def detect_beat_class_pattern(classes: list[int]) -> str | None:
    if len(classes) < 4:
        return None
    if all(c == 0 for c in classes):
        return "ALL_NORMAL"
    if all(c == 4 for c in classes):
        return "ALL_PACED"

    # Bigeminy: detect [0,1] or [0,2] pattern repeating for >= 4 pairs
    # Slide through all positions, not just even indices
    for pvc_class in (1, 2):
        best_run = 0
        run = 0
        for i in range(len(classes) - 1):
            if classes[i] == 0 and classes[i+1] == pvc_class:
                run += 1
                best_run = max(best_run, run)
            else:
                run = 0
        if best_run >= 4:
            label = "BIGEMINY_PVC" if pvc_class == 1 else "BIGEMINY_PAC"
            return label

    # Trigeminy: detect [0,0,1] repeating for >= 2 triplets
    for pvc_class in (1,):
        count = 0
        for i in range(len(classes) - 2):
            if classes[i] == 0 and classes[i+1] == 0 and classes[i+2] == pvc_class:
                count += 1
        if count >= 2:
            return "TRIGEMINY_PVC"

    ectopic_fraction = sum(1 for c in classes if c != 0) / len(classes)
    if ectopic_fraction < 0.1:
        return "OCCASIONAL_ECTOPY"
    if ectopic_fraction > 0.3:
        return "FREQUENT_ECTOPY"
    return None
```
