## EPM Adjudication

### Issue 1 — `filtfilt` Edge Transients for 0.05 Hz Highpass on Short Recordings
**RRC Severity:** CRITICAL
**EPM Decision:** FIXED IN DOCUMENT

`apply_highpass_filter()` docstring updated with an explicit edge transient warning: for a 0.05 Hz highpass Butterworth order 4 with filtfilt, edge transients from reflection padding affect approximately the first and last 3 seconds of the recording.

`compute_safe_analysis_window(n_samples, fs, lowcut_hz)` added — returns `(safe_analysis_start_sample, safe_analysis_end_sample)` using a 3-second conservative empirical margin, to be tightened by VT-1.2-02 empirical measurement.

`PreprocessedECGRecord` now includes `safe_analysis_start_sample: int` and `safe_analysis_end_sample: int` as typed fields. All downstream nodes (ECGdeli, feature extraction) must restrict analysis to this window. Issue resolved.

### Issue 2 — Wander Severity Heuristic Conflates QRS Amplitude with Wander
**RRC Severity:** REQUIRED
**EPM Decision:** FIXED IN DOCUMENT

The amplitude heuristic `max(|signal|) - percentile(|signal|, 10)` replaced with FFT-based low-frequency band power fraction:

```python
wander_mask = (fft_freqs >= 0.05) & (fft_freqs <= 0.5)
wander_power_fraction = sum(fft_mag[wander_mask]²) / sum(fft_mag[total]²)
if wander_power_fraction > 0.15:
    # use wavelet fallback
```

This directly measures the fraction of signal energy in the wander band, independent of QRS amplitude. Initial threshold 0.15 (15% of power in wander band) to be calibrated by VT-1.2-02. Issue resolved.

---

### Issue 3 — Raw Signal Sample Grid Misaligned with ECGdeli Fiducial Indices After Resampling
**RRC Severity:** REQUIRED
**EPM Decision:** FIXED IN DOCUMENT

`PreprocessedECGRecord` now includes `morphology_signal: np.ndarray` — raw amplitude at the 500 Hz sample grid. Construction:

- If source was 500 Hz: `morphology_signal = ecg_record.signal` (unmodified)
- If source required resampling: `morphology_signal = signal_before_filtering` (resampled to 500 Hz, no filters applied)

`signal_before_filtering = signal.copy()` captured after resampling and before any filter step. Node 1.5 morphology measurements use `morphology_signal`, not `raw.signal` and not `preprocessed_signal`. Downstream interface contract updated. Issue resolved.
