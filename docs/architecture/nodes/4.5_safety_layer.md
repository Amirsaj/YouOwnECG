## 1. Safety Layer Scope

The safety layer is NOT a content moderation filter for general LLM misuse. It serves a specific clinical safety function:

1. **Hallucination detection**: catch agent outputs that assert clinical facts contradicted by the retrieved evidence
2. **Out-of-scope detection**: catch agent outputs that go beyond the YouOwnECG domain (ECG interpretation)
3. **Disclaimer enforcement**: ensure all LLM-generated outputs carry required clinical disclaimers
4. **Confidence floor enforcement**: ensure STAT-level findings are never reported with suppressed confidence

The safety layer is applied to agent outputs before they reach the UI layer (Node 3.x).

## 2. Hallucination Detection

### 2.1 Entailment Check

For each factual claim in the agent's output, verify it is entailed by (or at least not contradicted by) the retrieved textbook chunks.

**Factual claim extraction:** The agent is prompted to output claims in a structured format:

```json
{
  "claims": [
    {
      "claim": "ST elevation > 1 mm in V2–V4 with upsloping morphology is consistent with early STEMI",
      "supporting_chunk_ids": ["chous_ch05_s5.2_p142_c0", "marriotts_ch08_s8.3_p211_c0"]
    }
  ]
}
```

**Entailment check:** Simple keyword-based verification (not a separate LLM call — latency budget):

```python
def check_claim_supported(claim: str, chunks: list[dict]) -> bool:
    """
    Returns True if the claim text shares significant keyword overlap with
    the cited chunks. False if citation is empty or keywords are absent.

    This is a lightweight check — not semantic inference. Catches the worst
    failure modes (no citation, citing wrong chapter) without adding latency.
    """
    if not chunks:
        return False

    claim_words = set(claim.lower().split())
    # Remove stop words
    stop_words = {"the", "a", "an", "in", "of", "with", "is", "are", "and", "or", "to", "for"}
    claim_words -= stop_words

    for chunk in chunks:
        chunk_words = set(chunk["text"].lower().split()) - stop_words
        overlap = len(claim_words & chunk_words) / max(len(claim_words), 1)
        if overlap >= 0.25:
            return True

    return False
```

When a claim fails the check, the claim is flagged and the output is amended with: `"[CITATION UNVERIFIED: this claim could not be confirmed against retrieved textbook evidence]"`.

### 2.2 Numerical Contradiction Check

Critical numerical thresholds used by agents are compared against the signal feature object. If the agent states a threshold that differs from the computed value by > 20%, it is flagged:

```python
AGENT_NUMERIC_FIELDS = [
    ("heart_rate_ventricular_bpm", "heart rate"),
    ("qrs_duration_global_ms", "QRS duration"),
    ("qtc_bazett_ms", "QTc"),
    ("pr_interval_ms", "PR interval"),
]

def check_numeric_consistency(agent_text: str, feature_object: dict) -> list[str]:
    """
    Returns list of inconsistency warnings (empty if all consistent).
    """
    warnings = []
    for field, label in AGENT_NUMERIC_FIELDS:
        computed_value = feature_object.get(field)
        if computed_value is None:
            continue
        # Extract numbers from agent text near the label
        pattern = rf"{label}[^\d]{{0,20}}(\d+(?:\.\d+)?)"
        matches = re.findall(pattern, agent_text, re.IGNORECASE)
        for match in matches:
            agent_value = float(match)
            if abs(agent_value - computed_value) / max(computed_value, 1) > 0.20:
                warnings.append(
                    f"{label}: agent stated {agent_value}, computed {computed_value}"
                )
    return warnings
```

## 3. Out-of-Scope Detection

The safety layer rejects agent outputs that contain responses outside ECG interpretation:

```python
OUT_OF_SCOPE_PATTERNS = [
    r"prescri(be|ption)",           # drug prescription
    r"(dosage|dose)\s+of\s+\w+",    # medication dosing
    r"recommend.{0,30}(surgery|procedure|catheter)",  # invasive procedure recommendations
    r"diagnos(e|is)\s+(the\s+)?patient",  # diagnosing the patient (not the ECG)
    r"call\s+(911|emergency|ambulance)",  # directing emergency action
]

def detect_out_of_scope(text: str) -> list[str]:
    flagged = []
    for pattern in OUT_OF_SCOPE_PATTERNS:
        if re.search(pattern, text, re.IGNORECASE):
            flagged.append(pattern)
    return flagged
```

Out-of-scope outputs are truncated at the flagged sentence and replaced with: `"[This analysis is limited to ECG interpretation. Clinical decisions require a qualified clinician.]"`

## 4. Disclaimer Enforcement

Every LLM-generated text block reaching the UI must contain a disclaimer. The safety layer appends it if absent:

```python
REQUIRED_DISCLAIMER = (
    "AI-GENERATED — NOT A CLINICAL DIAGNOSIS — CLINICAL CORRELATION REQUIRED"
)

def enforce_disclaimer(text: str) -> str:
    if "ai-generated" not in text.lower() and "not a clinical diagnosis" not in text.lower():
        return text + f"\n\n— {REQUIRED_DISCLAIMER}"
    return text
```

This is a last-resort enforcement. All agent system prompts already instruct the agent to include the disclaimer. This catches any case where the agent omits it.

## 5. Confidence Floor Enforcement

STAT-level findings cannot have their confidence suppressed or reported as NORMAL by any agent:

```python
STAT_CONDITIONS = {
    "stemi", "wellens", "de_winter", "vt_vf",
    "complete_avb", "hyperk_sine_wave", "brugada_type1", "long_qt_tdp_risk",
}

def enforce_confidence_floor(finding_type: str, agent_confidence: str, signal_confidence: str) -> str:
    """
    If signal pipeline detected a STAT condition, agent cannot downgrade it to NORMAL.
    Agent may disagree but disagreement is logged, not silently accepted.
    """
    if finding_type in STAT_CONDITIONS and agent_confidence == "NORMAL":
        # Log disagreement; return the higher of signal vs agent
        log_safety_event(
            event_type="STAT_DOWNGRADE_BLOCKED",
            finding=finding_type,
            signal_confidence=signal_confidence,
            agent_confidence=agent_confidence,
        )
        return signal_confidence  # keep signal pipeline's assessment
    return agent_confidence
```
