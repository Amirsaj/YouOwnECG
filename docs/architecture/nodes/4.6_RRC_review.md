## Issues Found

### REQUIRED-1: parse_agent_citations() silently discards out-of-range citation indices

**Location:** Node 4.6, Section 3

```python
for idx in cited_indices:
    if 0 <= idx < len(retrieved_chunks):
        citations.append(...)
```

If the agent cites `[7]` but only 5 chunks were retrieved, the citation is silently discarded. The finding then appears in the UI with fewer citations than the agent claimed, with no indication that the citation was invalid. In a cardiologist review context, a finding listed as "supported by 1 source" when the agent claimed 2 sources is misleading — it implies a source was lost rather than never existed.

**Required fix:** Out-of-range citation indices must be logged as a safety event and marked in the output:

```python
for idx in cited_indices:
    if 0 <= idx < len(retrieved_chunks):
        citations.append(...)
    else:
        log_safety_event(
            event_type="INVALID_CITATION_INDEX",
            details=f"Agent cited [{idx + 1}] but only {len(retrieved_chunks)} chunks retrieved",
            finding=claim_finding_type,
        )
        # Append a placeholder citation so the UI can display "[citation N: invalid]"
        citations.append(InvalidCitation(index=idx + 1))
```

### REQUIRED-2: No-citation fallback text does not distinguish "no relevant evidence" from "RAG not invoked"

**Location:** Node 4.6, Section 5

`NO_CITATION_TEXT` is displayed when no chunks are retrieved. But there are two distinct cases:
1. The RAG system was invoked, searched, and found no chunks above the similarity threshold (genuinely no relevant textbook evidence)
2. The RAG system was not invoked for this finding (e.g., the finding type is purely signal-based, like heart rate)

The current spec presents the same fallback text for both cases, which would mislead a cardiologist reviewing a finding about heart rate — they'd see "No textbook evidence retrieved" and wonder if the RAG system failed.

**Required fix:** Distinguish the two cases with separate fallback text:

```python
NO_CITATION_SEARCHED = (
    "No textbook evidence retrieved for this finding. "
    "The analysis is based on signal features alone."
)

NO_CITATION_NOT_SEARCHED = (
    "This finding is based on direct signal measurement. "
    "No textbook retrieval was performed."
)
```

Each `DiagnosticFinding` carries a boolean `rag_invoked: bool` to indicate which case applies.
