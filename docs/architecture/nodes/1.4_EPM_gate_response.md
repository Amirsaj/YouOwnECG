**RRC Severity:** CRITICAL
**EPM Decision:** FIXED IN DOCUMENT

### Issue 2 — AFib P-Wave Post-Processing Stated but Not Specified
**RRC Severity:** REQUIRED
**EPM Decision:** FIXED IN DOCUMENT

`_apply_condition_corrections(fpts_aligned, afib_suspected, rhythm_regularity_cv)` added as Section 4b. Explicitly:
- AFib suspected → Pon/Ppeak/Poff columns 0–2 set to -1 for all beats
- Irregular rhythm (CV > 0.20, non-AFib) → Pon/Poff set to -1; Ppeak preserved
- Called after `align_beats_across_leads()`, before `FiducialTable` construction — mandatory, not optional

Issue resolved.
