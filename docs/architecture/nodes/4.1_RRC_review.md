## Issues Found

### REQUIRED-1: sort_blocks_by_reading_order() uses a single midpoint heuristic that fails for asymmetric column layouts

**Location:** Node 4.1, Section 2.2

The two-column reconstruction splits blocks at `page_width * 0.9` midpoint. This works for symmetric two-column layouts where columns are equal width. However, Chou's Electrocardiography uses an asymmetric layout in some chapters where a narrow margin column (containing figure captions and callouts) occupies ~25% of the page width and the main text occupies 75%. The 90% midpoint threshold would assign all text to the left column and the narrow margin column would be sorted last — but since the margin column is narrow (< 10% of page), the heuristic would assign it to the left column and it would be interleaved randomly.

**Required fix:** The midpoint heuristic must be data-driven, not fixed at 90%. Detect the actual column split by clustering x-coordinates of left edges of all blocks:

```python
from sklearn.cluster import KMeans
import numpy as np

def detect_column_split(text_blocks: list[dict]) -> float | None:
    """
    Returns the x-coordinate split point if two columns detected, else None.
    """
    left_edges = np.array([b["bbox"][0] for b in text_blocks]).reshape(-1, 1)
    if len(left_edges) < 4:
        return None
    km = KMeans(n_clusters=2, n_init=10, random_state=0).fit(left_edges)
    centers = sorted(km.cluster_centers_.flatten())
    gap = centers[1] - centers[0]
    if gap < 30:  # less than 30px gap — likely single column
        return None
    return (centers[0] + centers[1]) / 2
```

### REQUIRED-2: extract_figures() extracts all images including decorative elements

**Location:** Node 4.1, Section 4.1

`extract_figures()` extracts all `type == 1` image blocks from every page. Medical textbooks contain decorative elements: publisher logos, section dividers, background textures, and small icons. These will be classified as DIAGRAM or PHOTO and embedded, polluting the figure index with irrelevant content.

**Required fix:** Filter extracted images by minimum size before processing:

```python
MIN_FIGURE_WIDTH_PX = 100
MIN_FIGURE_HEIGHT_PX = 80

# Apply in extract_figures():
if img_info.get("width", 0) < MIN_FIGURE_WIDTH_PX or img_info.get("height", 0) < MIN_FIGURE_HEIGHT_PX:
    continue  # skip decorative elements
```
