## 1. Cardiologist Reading Workflow

A cardiologist reading an ECG follows a systematic approach:
1. Rate and rhythm
2. Intervals (PR, QRS, QTc)
3. Axis
4. Morphology — P wave, QRS (voltage, morphology, progression), T waves, ST
5. Comparison to previous ECG (if available)
6. Clinical correlation

The cardiologist extension provides depth at each of these steps, beyond what the ER triage view offers.

---

## 2. Beat-by-Beat Comparison View

**Side-by-side beat overlay:** Multiple beats rendered on the same time axis, aligned at the R-peak. Each beat is a separate colored trace. Normal beats: gray; PVCs: orange; PACs: blue; most-abnormal beat: red.

This view reveals:
- Beat-to-beat ST variation (evolving STEMI)
- Alternans (even/odd amplitude difference)
- Rate-dependent QRS widening (wide at fast rate, narrow at slow)

**Trend chart below the overlay:** Line chart of PR, QRS, and QTc across beats (x = beat index, y = value in ms). Wenckebach pattern is immediately visible as a sawtooth on the PR line.

## 4. Reasoning Chain Viewer (XAI Interface)

The most clinically important feature of the cardiologist extension. Each diagnostic finding expands to show:

```
Finding: Possible Anterior STEMI [HIGH confidence]
  ├── Signal evidence
  │   ├── V1: ST elevation +0.08 mV (threshold: 2.0 mV) — NOT above threshold
  │   ├── V2: ST elevation +0.28 mV (threshold: 2.0 mV) — NOT above threshold
  │   ├── V3: ST elevation +0.31 mV (threshold: 2.0 mV) — NOT above threshold
  │   └── Note: values trending upward (progressive change detected)
  ├── Agent reasoning (IT agent, verbatim from reasoning trace)
  │   "The ST elevation in V2–V4, while individually below the 2.0 mV STEMI
  │    threshold, shows progressive upsloping morphology and a T/QRS ratio of
  │    0.82 in V3 (hyperacute T). The combination of upsloping ST + hyperacute T
  │    in 3 contiguous leads with progressive beat-to-beat increase is consistent
  │    with early anterior STEMI before full threshold elevation develops. The
  │    symmetric T-wave in V2–V3 (symmetric_inversion_v2v3 = False for now but
  │    trend is toward inversion) also warrants monitoring."
  ├── Cross-domain hints received: ["monitor_for_wellens_v2v3"]
  ├── VL2 visual: "st_elevation_leads: ['V2', 'V3'] — VL2 CONFIRMS ST changes"
  └── Textbook evidence
      "Chou's Electrocardiography, Ch. 8, p. 247: 'Hyperacute T waves with
       upsloping ST elevation represent the earliest phase of STEMI, often
       preceding diagnostic ST elevation by 30–60 minutes...'"
```

**Evidence trail:** Each finding links to:
- The exact feature values that drove it (clickable to highlight on ECG)
- The agent's verbatim reasoning chain (not a summary)
- The RAG citations with book/chapter/page

## 5. Morphology Comparison Mode

**Patient ECG vs textbook reference:**
- Left panel: patient's ECG waveform for the selected lead/beat
- Right panel: reference ECG waveform from textbook figure (embedded in the RAG system)
- Both rendered at identical scale
- Overlay option: semi-transparent superimposition

**Textbook reference selection:** The RAG system retrieves the highest-relevance ECG figure for the condition. The cardiologist can cycle through multiple reference figures.

**Previous ECG comparison:**
- If `ecg_id` belongs to a patient with prior ECGs in the local database, a "Compare with previous" button appears
- Previous ECG loads side-by-side with the same scale and lead arrangement
- Δ annotations: automatically computed ST delta, QTc delta between studies

## 6. Report Generation

### 6.1 Report Structure

```
YouOwnECG — AI-GENERATED ECG ANALYSIS REPORT
⚠ NOT A CLINICAL DIAGNOSIS — AI-GENERATED — CLINICAL CORRELATION REQUIRED ⚠

Patient:  [Sex], [Age] years
ECG date: [timestamp]
Recording: 10 seconds, 12-lead, 500 Hz

FINDINGS:
  1. [HIGH] Hyperacute T waves V2–V3, possible early anterior STEMI
     Evidence: ST +0.28–0.31 mV V2–V3; T/QRS ratio 0.82 V3; progressive upsloping
     Source: Chou's, Ch. 8, p. 247

  2. [MEDIUM] Borderline QTc (Bazett 448 ms, normal < 440 ms male)
     Evidence: QTc 448 ms (Bazett), 431 ms (Fridericia)
     Source: AHA Guidelines 2009

MEASUREMENTS:
  Heart rate: 71 bpm  |  PR: 162 ms  |  QRS: 94 ms  |  QTc: 448/431/441/443 ms
  Axis: +45° (Normal)  |  Beats: 9 normal, 2 PVC

QUALITY:
  8 leads GOOD, 3 ACCEPTABLE, 1 POOR (V3 — findings in V3 have reduced confidence)

AGENT REASONING: [Expandable section — agent chain-of-thought verbatim]

⚠ AI-GENERATED — THIS IS NOT A CLINICAL DIAGNOSIS
  This analysis was generated by an AI system and must be reviewed by a qualified
  clinician before any clinical decisions are made.
  YouOwnECG v1.0 | Pipeline: ecg_id [UUID]
```

### 6.2 Export

- PDF export: full report including measurement table, findings, citations, ECG image
- The `"AI-GENERATED"` label is rendered as a non-removable watermark on each page of the PDF
- The label cannot be removed by CSS or user action — it is part of the PDF canvas, not a DOM element

### 6.3 Cardiologist Annotation

Cardiologists can annotate/override findings before export:
- Each finding has an "Override" button → opens free-text annotation field
- Overridden findings are displayed with "Clinician override: [annotation]" prefix
- Overrides are logged to the audit trail
