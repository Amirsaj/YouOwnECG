## Resolution of RRC Issues

### REQUIRED-1: Float32Array serialization through Server Component boundary

**Fix applied to Node 3.5:**

ECG signal data is transferred as a base64-encoded string through the Server Component boundary and decoded client-side:

```typescript
// In Server Component (app/ecg/[ecgId]/page.tsx)
const signalRaw: Float32Array[] = await fetchECGSignal(ecgId);  // from FastAPI
const signalBase64: string[] = signalRaw.map(lead =>
    Buffer.from(lead.buffer).toString('base64')
);
// Pass signalBase64 (serializable string[]) to Client Component

// In Client Component (ECGCanvas.tsx)
function decodeSignal(base64: string): Float32Array {
    const binary = atob(base64);
    const bytes = new Uint8Array(binary.length);
    for (let i = 0; i < binary.length; i++) bytes[i] = binary.charCodeAt(i);
    return new Float32Array(bytes.buffer);
}
```

This is documented in Node 3.5 Section 2 as a required conversion step at the Server/Client boundary.

### REQUIRED-2: Q&A streaming timeout with AbortController

**Fix applied to Node 3.5:**

Q&A streaming uses `AbortController` with staged timeout messages:

```typescript
const QA_SLOW_THRESHOLD_MS = 5000;
const QA_ABORT_THRESHOLD_MS = 15000;

async function streamQAResponse(query: string, context: object) {
    const controller = new AbortController();
    let firstTokenReceived = false;

    const slowTimer = setTimeout(() => {
        if (!firstTokenReceived) setStatus("Response is taking longer than expected. Still working...");
    }, QA_SLOW_THRESHOLD_MS);

    const abortTimer = setTimeout(() => {
        if (!firstTokenReceived) {
            controller.abort();
            setStatus("The AI response timed out. The signal analysis is still available above.");
        }
    }, QA_ABORT_THRESHOLD_MS);

    try {
        const response = await fetch("/api/qa", {
            method: "POST",
            body: JSON.stringify({ query, context }),
            signal: controller.signal,
        });
        firstTokenReceived = true;
        clearTimeout(slowTimer);
        clearTimeout(abortTimer);
        // handle streaming response...
    } catch (e) {
        if (e.name !== "AbortError") throw e;
    }
}
```
